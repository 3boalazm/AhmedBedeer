# Ahmed M. Bedeer — Portfolio (Multi-Page)

A distinctive, dark-themed portfolio for **Ahmed M. Bedeer** — Enterprise Architect & Digital Transformation Specialist. Rebuilt from a single page into a **multi-page static site** and upgraded with Solutions-Architecture / Enterprise-Architecture best practices (case studies with trade-offs & ROI, governance, a skills matrix, leadership content, and a downloadable CV).

No build step, no framework, no dependencies — just static HTML/CSS/JS. Deploys to Vercel, Netlify, or GitHub Pages as-is.

## Shared assets

- `assets/styles.css` — the full design system (original CSS + new components). Edit once, applies everywhere.
- `assets/main.js` — reveal-on-scroll, mobile menu drawer, animated skill bars.
- `assets/Ahmed-M-Bedeer-CV.pdf` — generated 2-page CV. **Replace with your own** if you prefer.

## How to deploy (Vercel)

1. Push this folder to a GitHub repo (or drag-and-drop the folder/zip into Vercel).
2. Vercel auto-detects a static site — no settings needed. `index.html` is the entry point.
3. Done. Internal links are relative, so it works on any host or subpath.

## How to edit / regenerate

The HTML pages are generated by Python scripts in `_src/` to keep the nav and footer identical across pages.

```bash
cd _src
python3 build.py        # regenerates all *.html
python3 make_cv.py      # regenerates the CV PDF
```

Edit content in `_src/pages.py`, `_src/pages2.py`, `_src/pages3.py`; edit styling in `assets/styles.css`. If you prefer editing the HTML directly, you can — just keep the nav block consistent across pages (or re-run the build).

> ⚠️ **Verify the financials.** ROI/TCO/OPEX figures labelled *"modeled"* in the case studies are illustrative estimates. Replace them with your verified client numbers before publishing.

## Design notes

- Dark navy → blue/cyan aesthetic; **Syne** (display) + **Inter** (body) + **JetBrains Mono** (technical).
- All colors are CSS custom properties in `:root` at the top of `styles.css`.
- Fully responsive; reveal-on-scroll via IntersectionObserver.
- To use a real headshot, replace the `AB` initials in the `.about-avatar` / `.nav-avatar` with an `<img>`.
